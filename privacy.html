Privacy Policy

Last updated: March 10, 2026

The short version

HIITTrack does not collect, store, or transmit any of your personal data. Period.

Data Collection

HIITTrack does not collect any personal information. The app contains:

  • → No analytics or tracking SDKs
  • → No crash reporting services
  • → No advertising frameworks
  • → No third-party code that phones home

Data Storage

All workout data (history, settings, custom intervals) is stored locally on your device. Nothing is uploaded to any server. There are no user accounts and no cloud sync.

Third-Party Services

HIITTrack uses no third-party services. The app makes zero network requests.

Website

This website (hiittrack.com) is a static site hosted on GitHub Pages. It does not use cookies, analytics, or tracking scripts of any kind.
